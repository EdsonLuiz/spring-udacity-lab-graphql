# GraphQL Lab
Create a GraphQL API that returns a list of dogs from an embedded H2 in memory database.

## Dependencies
- Spring Boot DevTools
- H2 Database
- Spring Web Starter
- Spring Data JPA
- Graphql Spring Boot Starter
- Graphql Java Tools
